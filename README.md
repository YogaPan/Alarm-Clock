# Alarm Clock

This is an 'electron + webpack + react' example for this slide:
https://hackmd.io/p/rJpxFxuu

## How to start?

### clone this project
```shell
$ git clone https://github.com/YogaPan/Alarm-Clock.git
```

### Install dependencies
```shell
$ npm install
```

### Start development
```shell
$ npm start
```

### Build your application
```shell
$ npm run dist
```
